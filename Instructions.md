Task 1: Set Up Git and GitHub
 Install Git on your computer.
 Configure your Git username and email.
 Create a GitHub account (if you don’t already have one).
Submit a screenshot showing your Git configuration (git config --list).
Task 2: Create a Local Repository
 Create a new project folder named data-analytics-version-control.
 Initialize it as a Git repository using git init.
 Add a sample Python or R script (e.g., data_cleaning.py or analysis.R) and a
README.md file.
 Make your first commit with a meaningful message.
Submit screenshots of the terminal showing git status and git log.
Task 3: Push to GitHub
 Create a new public repository on GitHub with the same name.
 Connect your local repository to the remote one (git remote add origin …).
 Push your local commits to GitHub.
